Please save your screenshots with the following filenames in this directory:

### System Overview

### Core Modules
- `dashboard.png` - Main dashboard with navigation sidebar

### Reports & Output
- `payroll-report.png` - Payroll report with transaction details
- `attendance-report.png` - Attendance report with period analytics
- `salary-slip.png` - Generated salary slip output
- `system-docs.png` - System documentation (optional)

## Image Requirements
- **Format**: PNG or JPG
- **Quality**: High resolution (minimum 1920x1080)
- **Optimization**: Compress for web (under 500KB per image)
- **Naming**: Use exact filenames as listed above

## How to Add Images
1. Save your screenshots with the exact filenames listed above
2. Place them in this `screenshots/hr_payroll/` directory
3. The HTML will automatically display them with professional styling
4. Images will have hover effects and responsive design
